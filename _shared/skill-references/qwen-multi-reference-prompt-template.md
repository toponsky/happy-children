# Qwen Image Edit 2511 三图参考提示词模板

基于实战验证的中文提示词格式，每个图像引用显式指定在画面中的位置和角色。

## 模板结构

```
图像3的[场景类型]上，图像1中的[角色1]在[位置]，[动作/姿态]，[关键道具]。远处/近处 图像2中的[角色2]在[位置]，[状态]，[关键约束]。角色不在错误位置。保持图像1[角色1]面部和[服装]。保持图像2[角色2]面部、衣服和[特征]。完全保留图像3[场景]背景。
```

## 已验证示例

### 街道场景 - 两人物 + 背景
```
图像3的街道上，图像1中的小男孩站在人行道上，背对镜头看向前方。远处前方图像2中的老奶奶坐在轮椅上，不戴眼镜。小男孩不在轮椅上。保持图像1小男孩面部和红色卫衣。保持图像2老奶奶面部、衣服和裸眼。完全保留图像3街道背景。
```

### 街道场景 - 奔跑动态
```
图像3的街道上，图像1中的小男孩背对镜头，正向前方奋力奔跑，双臂摆动，红色书包在背上跳动。远处前方图像2中的老奶奶坐在轮椅上，不戴眼镜，面朝镜头方向。小男孩不在轮椅上。保持图像1小男孩面部和红色卫衣。保持图像2老奶奶面部、衣服和裸眼。完全保留图像3街道背景。
```

### 街道场景 - 单人挺直站姿（说话/敬礼）
```
图像3的街道上，图像1中的爱尔兰警察Shane身穿深绿色警服，笔直地站在街道中央，正面朝向镜头。腰背挺直，站姿端正挺拔，双肩平齐，下巴微收。脸上带着温暖微笑，正在对着镜头说话。他的左手自然垂在身体一侧，右手尚未举起。身体没有前倾没有弯腰。保持图像1警察Shane的面部特征和深绿色警服颜色款式。完全保留图像3街道背景的路面、树木、房屋和光线。
```

### 街道场景 - 远景单人（远处走向镜头）
```
图像3的街道远景，图像1中的爱尔兰警察Shane身穿深绿色警服，站在远处街道的中央，小小的身影，正面朝向镜头方向，正在向镜头走来，步伐坚定。这是远景镜头，警察在画面中占比很小，周围是安静的住宅区街道、树木和房屋。保持图像1警察Shane的深绿色警服。完全保留图像3街道背景的路面、树木、房屋和光线。
```

## 关键规则

1. **显式否定错误状态**：如果某个角色不应出现在某位置，明确写"小男孩不在轮椅上"
2. **三句"保持"收尾**：保持图像1[角色+服装]、保持图像2[角色+特征]、完全保留图像3[背景]
3. **参考图编号对应**：image1=主角(R1)、image2=配角(R2)、image3=背景(R7)
4. **面部约束不可省**："保持面部"防止 Qwen 重绘时改变角色长相
5. **Denoise 0.4-0.5**：太低参考图融合不够，太高偏离参考图
