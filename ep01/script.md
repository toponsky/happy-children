# 🎬 Happy Children — 第一集：上学的路上

## 剧情概要

清晨，Michael背着书包去上学。路上看到王奶奶坐在轮椅上，她的眼镜掉在地上，她伸手够不着。Michael赶紧跑过去，捡起眼镜递给王奶奶。王奶奶戴上眼镜，露出感激的笑容。Michael推着她送回家，告别后高高兴兴跑向学校。
